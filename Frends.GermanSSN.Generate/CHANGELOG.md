# Changelog

## [1.0.0] - 2025-11-17

### Added

- Initial implementation of German Social Security Number generation
- Support for deterministic SSN generation based on person data
- Input validation with three modes: None, Basic, and Strict
- Flexible output formatting (with/without spaces)
- Comprehensive error handling with custom error messages
- SSN component breakdown for transparency
- Check digit calculation using modified Luhn algorithm
- Full unit test coverage including validation and error handling scenarios
- XML documentation for all public APIs
